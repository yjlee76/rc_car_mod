# RC Car Mod Project
This is the project repository for a RC car-based autonomous driving course.
The used hardware includes the following
	1. NVIDIA Jetson Orin Nano or NX
	2. USB camera
	3. Slamtec C1 lidar
	4. Yahboom ROS robot control board V3
	5. Yahboom 20 kg cm servo (for steering)
	6. Yahboom 520 DC Gear motor (for driving)

src directory contains ROS 2 Jazzy packages for the project.
